== Description ==

Spa and Salon is a beautifully designed and easy to use WordPress theme. It can be used for SPA, Salon Business and several other kinds of business. Spa and Salon Theme is fully responsive. So no matter what device your users are on, it will always look great!


== Installation ==
        
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Documentation for Home Page Setup ==

A. Banner Section

  1. Go to Appearance> Customize>Homepage Settings> Banner Section
  2. Check Enable Banner Section  to display it on the Homepage.
  3. Select Banner Post. The Post title will appear as the banner title and the post excerpt will appear as the description.  The featured image of the selected Post will be displayed as the banner image.
  4. Enter Readmore Text.
  5. Click Save & Publish.

B. Featured Section

  1. Go to Appearance> Customize>Homepage Settings> Featured Section 
  2. Check Enable Featured Posts Section to display the section in the homepage.
  3. Select Featured Post One, Post Two and Post Three. 
  4. Enter Favicon name for Post One, Post Two and Post Three. 

  Note: To find the Favicon icon name go to :  http://fontawesome.io/icons/
  Find the Icon that suits your website. For example You like the icon with the name wpbeginner , then enter wpbeginner as Favicon name

  5. Click Save & Publish.

C. About Section

  1. Go to Appearance> Customize>Homepage Settings> About Section
  2. Check Enable About Section  to display it on the Homepage.
  3. Select About Section Post. The Post title will appear as the banner title and the post excerpt will appear as the content.  The featured image of the selected Post will be displayed as image on the About Section.
  4. Enter Readmore Text.
  5. Click Save & Publish.

D. Service Section

  1. Go to Appearance> Customize>Homepage Settings> Service Section
  2. Check Enable About Section  to display it on the Homepage.
  3. Enter Service Section Title  and Service Section content.
  4. Select Service Post One, Post Two, Post Three, Post Four, Post Five and Post Six.  
  5. Click Save and Publish.

E. Testimonial Section :

  1. Go to Appearance> Customize>Homepage Settings> Testimonial Section
  2. Check Enable Testimonial Section  to display it on the Homepage.
  3. Enter Testimonial Section Title  and Testimonial Section content.
  4. Select the category of the Posts to display on Testimonial Section. The Posts title under the selected category  will appear as the cutomersâ€™ name and the respective featured images of the Posts will be displayed under the section.  
  5. Click Save and Publish.
